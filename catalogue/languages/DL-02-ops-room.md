# DL-02 . KATA (formerly OPS ROOM)

Motto: Perfect through repetition. (Japan: discipline and continuous practice.)

The teal workhorse for recurring status reporting: weekly decks, PMO
packs, progress dashboards. Dense but gridded; one teal highlight per
chart, context in pale teal or gray.

**Canonical tokens: [`tokens/DL-02.json`](../../tokens/DL-02.json).**
The JSON file is the single source of truth; this page is commentary.

## Signature

Header band with a period stamp, a KPI strip (5 cards max), status chips
that always carry the status word. Charts: primary teal `#008080`, one
highlight per chart, S-curves for schedule, stacked bars for pipelines,
heatmaps for readiness.

## Chrome

No kicker, no eyebrow (`chrome: {kicker: false, eyebrow: false}`).
Titles sit at the default y=56.

## Banned in this language

More than 5 KPI cards, zebra striping, vertical table borders, decorated
headers, more than one saturated series per chart.

## QA additions

- Every status chip carries a word, not just a color.
- KPI strip <= 5 cards.
