Charts / CH-PTW-06
CH-PTW-06 . family PTW

WAFFLE

10x10 unit grid, one share. Needs: one percentage of one whole; the most honest single-share visual.

sharepart-to-whole

Use when

  • one share communicated with countable units: 62 of every 100
  • audiences who distrust abstract charts (units are literal)

Do not use when

  • multiple shares (stacked bar or pie)
  • shares under 3% (invisible cells)

See instead

  • more than one share: CH-PTW-01

Alternatives

  • CH-PTW-03 (two or three shares)

Rules

  • exactly 100 cells; filled count = rounded percent
  • fill bottom-up so the grid reads like a container
  • the big number restates the percent beside the grid

Never do this

  • icon waffles (pictogram people)
  • multi-color waffles
  • partial cells

QA before delivering

  • cell count = 100
  • filled = round(percent)
  • big number matches
  • source present

Data shape
